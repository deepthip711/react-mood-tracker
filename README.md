# React Mood Tracker

An interactive Mood Tracker built using **React** that allows users to select their current mood and dynamically updates the user interface using React state. The application demonstrates basic state management, conditional rendering, and event handling in React.

🌐 **Live Website:**
https://deepthip711.github.io/react-mood-tracker

---

## Features

* Select different moods (Happy, Neutral, Sad, Angry)
* Dynamic background color changes based on selected mood
* Emoji display representing each mood
* Responsive and clean user interface
* Interactive buttons with hover effects
* Built using React Hooks (`useState`)

## How to Run Locally

1. Clone the repository

```
git clone https://github.com/deepthip711/react-mood-tracker.git
```

2. Navigate into the project folder

```
cd react-mood-tracker
```

3. Install dependencies

```
npm install
```

4. Start the development server

```
npm start
```

The app will run at:

```
http://localhost:3000
```
